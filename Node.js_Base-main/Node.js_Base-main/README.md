## API(Application Programming Interface)
- 어플리케이션을 프로그래밍하기 위해서 제공되는 인터페이스
- 인터페이스란, fs.readfile();과 같이 함수를 만든 개발자와 사용하는 개발자 간의 약속한 조작 소스코드이다.
- 개발자들은 어플리케이션을 원하는대로 조작하기 위해서 여러 API를 사용하여 개발한다.
